Bootstrap Assignment

1.What are the advantages of Bootstrap?

Ans => Bootstrap is open-source, making it an accessible choice for developers aiming to create responsive and visually appealing websites.

  1. Bootstrap’s ready-made components accelerate development, enabling faster project completion.
  2. The built-in grid system ensures websites adapt seamlessly to diverse screen sizes and devices.
  3. Bootstrap maintains a uniform look and feel across various elements, ensuring a cohesive user experience.
  4. Bootstrap handles browser differences, minimizing the need for extensive testing and ensuring a consistent user experience.
  5. Customizable themes, variables, and mixins provide flexibility to match specific project requirements and branding.


2.What is a Bootstrap Container, and how does it work?

Ans => Containers are the most basic layout element in Bootstrap and are required when using our default grid system. Containers are used to contain, pad, and (sometimes)
center the content within them. While containers can be nested, most layouts do not require a nested container.

4.What do you know about the Bootstrap Grid System?

Ans => Bootstrap Grid System allows up to 12 columns across the page. You can use each of them individually or merge them together for wider columns. You can use all
combinations of values summing up to 12. You can use 12 columns each of width 1, or use 4 columns each of width 3 or any other combination.

6.What is a Button Group, and what is the class for a basic Button Group?

Ans => Button Groups in Bootstrap is a class of name 'btn-group' which is used to create a series of buttons in groups (without spaces) vertically or horizontally.
This is the basic syntax of the button group class where each button has its own class of 'btn'.

Ans =>

Bootstrap to make thumbnails Step by step.

  1. Step: Include Bootstrap and jQuery CDN into the < head> tag before all other stylesheets to load ourCSS.
  2. Step: Add < div> tag in the HTML body with class row. In that < div> create four div sections to createfour images.

8.In Bootstrap 4, what is flexbox?

Ans => In Bootstrap 4, flex box is used to control the layout and alignment specification of Bootstrap 4 components. This box makes it easier to design flexible responsive layout
structures without using float or positioning attributes.


9.How can one create an alert in Bootstrap?

Ans =>

Create an alert in Bootstrap

  1. Step : Include Bootstrap and jQuery CDN into the < head > tag before all other stylesheets to load our CSS.
  2. Step : Add the .alert and the warning alert contextual classes (e.g., .alert-warning).
Example :- < div class="alert alert-warning" role="alert" > A simple warning alert—check it out! < /div >


10.What is a bootstrap card and how would you create one?

Ans => A card is a flexible and extensible content container. It includes options for headers and footers, a wide variety of content, contextual background colors, and powerful display options.
It replaces the use of panels, wells and thumbnails. All of it can be used in a single container called card.
